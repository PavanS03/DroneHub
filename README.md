 **Modern Android-Based Drone Rental & Mission Management Platform**

**DroneHub** is a modern Android-based **Drone Rental & Mission Management Platform** built using **Java** and **Firebase Realtime Database**.

The platform streamlines **drone booking**, **operator assignment**, **mission execution**, **payment handling**, and **real-time workflow management** through dedicated **Admin**, **User**, and **Operator** modules.

This project was developed as a collaborative team project focused on simulating a real-world **enterprise drone operations ecosystem** with **scalable workflow management** and **live tracking capabilities**.



# 📌 𝗢𝘃𝗲𝗿𝘃𝗶𝗲𝘄

Managing drone operations manually can be complex and inefficient. **DroneHub** solves this problem by providing a **centralized platform** where users can **book drones**, admins can **manage operations**, and operators can **execute missions seamlessly** through **real-time synchronization** and **smart workflow handling**.

The system is designed with a modern **Android architecture** and **Firebase-powered backend** to ensure **responsive performance**, **secure access control**, and **dynamic real-time updates**.

##  Clone the Repository

git clone https://github.com/PavanS03/DroneHub.git


##  Navigate to the Project Directory

cd DroneHub


##  Open the Project in Android Studio

- Launch **Android Studio**
- Open the cloned project folder

##  Configure Firebase

- Create a **Firebase project**
- Add google-services.json
- Enable **Firebase Authentication**
- Enable **Firebase Realtime Database**

##  Sync Gradle Files & Run the Application

- Sync Gradle
- Build and run the project
